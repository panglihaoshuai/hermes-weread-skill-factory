# Privacy（隐私）

---

## 概述

weread-to-skill-factory 是一个本地 Hermes skill factory，不做平台，不上传用户数据。

---

## 隐私原则

### 1. 本地处理

所有数据处理在本地完成，不上传到任何服务器。

### 2. 不上传用户数据

不上传用户读书笔记、划线、批注、Viva 回答等个人数据。

### 3. 不包含真实 WeRead 数据

本项目不包含真实 WeRead 数据，只包含虚构示例。

### 4. 用户需自行安装 weread skill

用户需要自行安装 weread skill，并提供自己的 WeRead API key。

### 5. 未通过 Reading Gate 不生成 book-skill

只有通过 Reading Gate 判定的书才能生成 book-skill，确保用户真正精读过。

---

## 数据来源

### WeRead 数据

- **来源**：微信读书 API
- **类型**：划线、批注、章节目录、阅读进度
- **存储**：本地文件系统
- **上传**：无

### Viva 回答

- **来源**：用户口头回答
- **类型**：对书籍的理解、思考、应用
- **存储**：本地文件系统
- **上传**：无

---

## 数据使用

### 用于 skill 生成

- WeRead 数据用于 Reading Gate 判定
- WeRead 数据用于 skill 蒸馏
- Viva 回答用于 skill 蒸馏

### 不用于其他目的

- 不用于广告
- 不用于用户画像
- 不用于数据交易

---

## 安全措施

### 1. 本地存储

所有数据存储在本地文件系统，不上传到云端。

### 2. 访问控制

只有用户本人可以访问自己的数据。

### 3. 数据删除

用户可以随时删除自己的数据。

---

## 用户责任

### 1. 保护 API key

用户需要保护自己的 WeRead API key，不要泄露给他人。

### 2. 备份数据

用户需要备份自己的数据，防止数据丢失。

### 3. 合规使用

用户需要合规使用本项目，不要用于非法用途。
