# 女娲蒸馏模式参考（nuwa-skill）

来源：`alchaincyf/nuwa-skill`（23k stars）
用途：参考其蒸馏思想，但不照抄。女娲蒸馏人物，我们蒸馏书籍。

## 五层蒸馏法

| 层 | 问题 | 书籍中的对应 |
|---|------|-------------|
| **表达 DNA** | 作者怎么说？ | 语言风格、类比模式、叙事结构 |
| **心智模型** | 作者怎么看世界？ | 核心框架、认知透镜 |
| **决策启发式** | 作者怎么判断？ | if-then 规则、权衡逻辑 |
| **反模式** | 作者绝对不会做什么？ | 价值观底线、明确警告 |
| **诚实边界** | 这个框架不能做什么？ | 局限、盲点 |

## 与女娲的关键差异

| 维度 | 女娲（人物蒸馏） | 我们（书籍蒸馏） |
|------|-----------------|-----------------|
| 数据来源 | 公开资料（推特/访谈/博客） | 微信读书（个人划线+批注+热门划线） |
| 个性化程度 | 通用（任何人可以蒸馏任何人） | 高度个性化（基于你的划线和批注） |
| 时间维度 | 实时（人还在说话） | 固定（书出版后不变） |
| 证据类型 | 公开言行 | 你认可的段落 + 群体热门 |
| 输出 | 人物视角 skill | 认知框架 skill |

## 蒸馏原则（从女娲继承）

1. **提取 HOW they think，不是 WHAT they said**
2. **心智模型 3-7 个是甜区** — 超过 10 个说明没综合
3. **决策启发式必须可执行** — "想清楚" 不是启发式，"反转问题" 才是
4. **必须标注诚实边界** — 一个不告诉你局限的 skill 不值得信任
5. **引用 ≠ 蒸馏** — 大段抄书不是蒸馏，提取框架才是
