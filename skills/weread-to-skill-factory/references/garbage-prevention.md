# Skill Garbage Prevention Rules（垃圾场防护规则）

以下红线在任何情况下不得违反：

1. **不得因为一本书出名就生成 book-skill**
2. **不得因为一本书在用户微信读书书架里就生成 book-skill**
3. **不得仅凭热门划线、公开点评、书籍简介生成 book-skill**
4. **不得根据短视频、排行榜、他人推荐、营销号总结生成 book-skill**
5. **用户没有个人阅读证据时，必须通过私下精读口试**
6. **证据不足时，只能生成 reading-card，不得生成 skill**
7. **如果一本书无法提取出可复用的判断规则、反模式、诊断问题和行动协议，不得生成 skill**
8. **如果新 book-skill 与已有 skill 高度重复，优先建议合并或扩展旧 skill，不得制造重复 skill**
9. **每个正式 book-skill 必须有 eval 测试**
10. **每个正式 book-skill 必须有不适用场景和反过度套用说明**
